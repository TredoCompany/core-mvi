# core-mvi-android
Tredo MVI Library based on Kotlin Flow

## **Использование**

build.gradle(root)
```groovy
allprojects {
    repositories {
        maven { url 'https://jitpack.io' }
    }
}
```

build.gradle(app)
```groovy
dependencies {
    implementation 'com.github.TredoCompany:core-mvi-android:1.0.0'
}
```
